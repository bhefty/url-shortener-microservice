API Basejump: URL Shortener Microservice

User Stories:
  • I can pass a URL as a parameter and I will receive a shortened URL in the JSON response.
  • When I visit that shortened URL, it will redirect me to my original link.

Example Creation Usage:

https://evening-peak-54321.herokuapp.com/new/https://www.freecodecamp.com http://evening-peak-54321.herokuapp.com/9130

Example Creation Output:

{ "original_url": "https://www.freecodecamp.com", "short_url": "http://evening-peak-54321.herokuapp.com/2703" }
